# voice-video-labs-releases
Official macOS beta releases of Voice &amp; Video Labs.
VOICE & VIDEO LABS
Official macOS Public Beta Releases

Voice & Video Labs is a macOS production application for narration, audio
processing, recording, visual asset management, and long-form video creation.

PUBLIC BETA

This repository provides official beta downloads of Voice & Video Labs.

Voice & Video Labs is currently beta software. Back up important files and
verify all generated output before using it in production.

SYSTEM REQUIREMENTS

- macOS
- Apple Silicon Mac
- Sufficient free disk space for local narration models, temporary media,
  and rendered output
- Internet connection for Azure Speech or ElevenLabs features

NARRATION PROVIDERS

- Kokoro local narration is included.
- Azure Speech requires the user's own Azure credentials.
- ElevenLabs requires the user's own ElevenLabs credentials.
- Azure and ElevenLabs usage charges are separate and are paid directly to
  those providers.
- No Azure or ElevenLabs credentials are included with the application.

MAJOR FEATURES

- DOCX and EPUB narration workflows
- Sentence-safe narration segmentation
- Local Kokoro text-to-speech
- Azure Speech integration
- ElevenLabs integration
- Voice previews and narration settings
- Narration checkpointing and recovery
- Audio conversion
- Audio merging and splitting
- Audio extraction
- Audio cleanup and enhancement
- Recording tools
- Overlay and mixing tools
- Visual prompt and image assignment workflows
- Video Production Lab
- FFmpeg-based video rendering
- Production Wizard
- Saved production sessions and recovery
- Metadata and output management

INSTALLATION

1. Download the latest macOS release from the Releases section.
2. Open the downloaded DMG or ZIP file.
3. Move Voice & Video Labs to the Applications folder if prompted.
4. Open Voice & Video Labs.
5. macOS may verify the signed and notarized application during first launch.

DOWNLOAD SAFETY

Only download Voice & Video Labs from this official repository or an official
Voice & Video Labs website.

Release pages may include a SHA-256 checksum so users can verify the downloaded
installer.

BETA TESTING

Please report:

- Application crashes
- Failed narration jobs
- Failed video renders
- Incorrect output
- Problems reopening saved projects
- Missing files
- Interface problems
- Unexpected provider errors
- Installation or launch failures

When reporting a problem, include:

- Voice & Video Labs version
- macOS version
- Mac model and processor
- Narration provider used
- A description of the steps that caused the problem
- Relevant error text or logs

Do not post any of the following in public issue reports:

- API keys
- Passwords
- Azure credentials
- ElevenLabs credentials
- Private scripts
- Personal documents
- Confidential project files
